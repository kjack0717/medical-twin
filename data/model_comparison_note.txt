[모델 선택 분석: Random Forest vs MLP]

데이터: 500명 가상환자, 3-클래스 독성 분류 (standard / dose_adjust / toxic)

[교차검증 F1-macro (5-fold)]
  RF  : 0.8145 +/- 0.0679  (폴드별: 0.776, 0.838, 0.938, 0.761, 0.760)
  MLP : 0.4618 +/- 0.0490  (폴드별: 0.539, 0.436, 0.447, 0.396, 0.491)

[테스트 셋 F1-macro]
  RF  = 0.7457,  MLP = 0.3939

[모델 선택 논거]
1. 안정성: RF의 CV 표준편차(0.0679)가 MLP(0.0490)보다 높으나 이는 소규모 불균형 데이터 특성입니다.
2. 해석 가능성: RF는 feature_importances_ 를 제공하여 임상 인자(체중·eGFR·유전형)의
   기여도를 정량화할 수 있습니다. MLP는 블랙박스 구조로 해석이 어렵습니다.
3. 소규모 불균형 데이터: n=500에서 MLP는 hyperparameter 민감도가 높아 과적합
   위험이 있습니다. RF의 앙상블 구조(200 trees)는 이를 완화합니다.
4. 클래스 불균형 처리: RF class_weight='balanced' 설정으로 희소 클래스(toxic, ~4%)
   탐지 성능을 명시적으로 강화했습니다.

=> 최종 채택 모델: RF (테스트 F1 기준)
